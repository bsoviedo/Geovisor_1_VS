Geovisor datos

En este visor de datos geográficos se presentan datos recolectados por el proyecto vidas sileciadas, los cuales fueron bajados desde su perfil en la red social twitter. El objetivo es mostrar, mediante el uso de información geográfica la localización de distintos hechos reportados por el protecto, la cual se centra principalmente en desapariciones forzadas.

Los datos se encuentran agrupados para una mejor visualización, en total se recuperaron al rededor de 2500 registros, por lo que, para acceder a cada uno de ellos es necesario hacer zoom en las distintas áreas que sean de interés. Cada punto desplegará el tweet por el cual fue localizado y el enlace para acceder al mismo.

Este geovisor se desarrolla como un ejercicio meramente acádemico , como parte de práctica para el uso de javascript en desarrollo web y su integración con datos y servicios de sistemas de información geográfica (SIG). No se obtiene ningún tipo de lucro con la información desplegada, la cual, es parte del proyecto vidas silenciadas.
